# Contributing

Thanks for helping improve `parkingpage-comingsoon`.

## Development

Use Node.js 24 LTS and npm.

```bash
npm ci
npm run dev
```

Before opening a pull request, run:

```bash
npm run lint
npm run typecheck
npm run build
```

## Pull Requests

- Keep changes focused and explain the user-facing effect.
- Include screenshots or a short screen recording for visual changes.
- Update README/docs when behavior, setup, deployment, or customization steps change.
- Do not commit local `.env` files, generated `dist/` output, or `node_modules/`.

## Issues

Please include the browser, operating system, reproduction steps, expected behavior, and actual behavior.
